# Smart Waste Monitoring System

## SDG

SDG 11 – Sustainable Cities

## Description

This project simulates IoT-based smart waste bins and sends data to cloud.
A dashboard visualizes data and alerts when bins are full.

## Features

- Real-time monitoring
- Cloud integration (ThingSpeak)
- Alerts system
- Trend prediction

## Tech Stack

- Python
- Streamlit
- ThingSpeak

## How to Run

1. Install requirements
2. Run simulate.py
3. Run Streamlit app
